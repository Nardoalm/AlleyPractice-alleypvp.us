# KaosPractice

**Autor:** [ysubz](https://knowplugins.com)  
**Website:** [knowplugins.com](https://knowplugins.com)

KaosPractice e um core de Practice PvP para Minecraft 1.8.

## Instalacao

1. Clone este repositorio.
2. Compile o projeto na sua IDE para gerar `KaosPractice.jar`.
3. Coloque `KaosPractice.jar` na pasta `plugins` do servidor.
4. Inicie o servidor e configure `database.yml`.
5. Reinicie o servidor.

## Uso

Configure os arquivos em `plugins/KaosPractice` para ajustar kits, arenas, visuais e permissoes.
